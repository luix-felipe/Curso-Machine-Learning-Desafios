# Curso Machine Learning - Desafios

Workspace separado contendo somente os desafios do curso.

Os scripts seguem o roteiro dos notebooks de desafio, com as lacunas preenchidas conforme os gabaritos do curso.

## Como usar

```powershell
pip install -r requirements.txt
python desafios\desafio_01\desafio_1.py
python desafios\desafio_02\desafio_2.py
python desafios\desafio_03\desafio_3.py
python desafios\desafio_04\desafio_4.py
```

Observacoes:

- O desafio 2 espera o arquivo `openface_nn4.small2.v1.t7` dentro de `desafios/desafio_02`.
- O desafio 3 espera o arquivo `video.mov` dentro de `desafios/desafio_03`.
